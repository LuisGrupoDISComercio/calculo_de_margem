Redefinição de senha
Para redefinir sua senha, entre em contato com o administrador do sistema.

O administrador irá gerar uma senha temporária e enviar para você.

Após receber, acesse o sistema e troque sua senha usando o botão 🔑 Senha no canto superior direito da tela.
Vincular Telegram

Passo 1: O usuário deve abrir o Telegram e buscar @DIS_Simulator_Bot
Passo 2: Enviar o comando /start
Passo 3: O bot vai responder com um código de 6 dígitos
Passo 4: O usuário informa o código abaixo

Para obter o Chat ID: o bot responde ao /start com o ID. Ou use @userinfobot no Telegram.

Redefinir Senha

Defina uma senha temporária e comunique ao usuário por WhatsApp. Ele deverá trocá-la após o login usando o botão 🔑 Senha.

Alterar Senha
